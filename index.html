SANAD App

SANAD app is a hybrid development environment app, that can be executed on both iOS and Android devices, developed using React Native which is an open-source mobile application framework created by Facebook. For the backend, we use Laravel which is a free, open-source PHP web framework we integrate with the MySQL database where we can store our data, then we run it on a real server using the MySQL workbench so we can have multiple access rather the local host. The application is intended to help the physiotherapist to set a special treatment plan for each patient by using videos, specific iteration for each exercise and direct message as well.
